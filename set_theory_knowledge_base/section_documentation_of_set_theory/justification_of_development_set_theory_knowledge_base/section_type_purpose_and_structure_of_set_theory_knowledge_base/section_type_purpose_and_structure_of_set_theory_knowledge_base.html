Интеллектулаьная система по теории множеств является базой из семейства баз, разработанных по технологии OSTIS. Её назначением является обучение и консультация студентов, программистов и всех желающих, решение задач по теории множеств. Система состоит из самой базы знаний, интеллектуального интерфейса,а также машины обработки знаний.
